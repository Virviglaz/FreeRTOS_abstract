Detailed Description

template<class T, size_t size = 2, const bool support_emplace = false>
class FreeRTOS::Queue< T, size, support_emplace >

Creates a new FreeRTOS queue instance.

@Note This is MPSC queue (multi producers, single consumer queue). If you need to use MPMC queue, you have to block the kernel until popped pointer is processed or copied.

Template Parameters
TType of object to be enqueued.
sizeMaximum queue size (maximum amount of object to store).
support_emplaceEnable support emplace constructor.
